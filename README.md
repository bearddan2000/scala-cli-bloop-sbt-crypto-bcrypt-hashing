# scala-cli-bloop-sbt-crypto-bcrypt-hashing

## Description
A demo for Bcrypt one-way hashing of
a password.

Compiled and ran from build server `bloop`.

# Build note
Dependencies must be compatable with jdk8 or less.

## Tech stack
- bloop
- bloop-sbt
- scala

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
